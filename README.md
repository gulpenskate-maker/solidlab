# solidlab

Nordic technology studio — marketing site for **solidlab.ai**.

## Stack

Static HTML + React (via Babel standalone) + CSS. No build step.

- `index.html` — entry point
- `site.css` / `surfaces.css` — styles
- `components.jsx` / `surfaces.jsx` — React components
- `colors_and_type.css` — design tokens (Nordic palette)
- `fonts/` — Geist + Geist Mono (variable)
- `assets/` — logos, brand guidelines PDF

## Brand

- **Palette:** Jotun-anchored Nordic (Oslo #4A5A62, Sort #0A0A0A, Antikkhvit #F4F1EA)
- **Typography:** Geist + Geist Mono
- **Signature element:** The forward slash `/`
- **Voice:** Building solid software. From the Nordic coast.

## Deploy

Auto-deploys to Vercel on push to `main`.

Production domain: `solidlab.ai`
